# BrandKit - Brand Asset Generator

**BrandKit** is a web-based automated design tool that allows businesses to centralize their brand identity and instantly generate professional collateral. It functions as a dynamic "Brand Toolkit," bridging the gap between raw business data and production-ready design assets.

## Core Features

### 1. Brand Setup (Configuration)
The entry point for defining the visual identity. All generated assets dynamically inherit these settings.
* **Business Profile:** Input fields for Business Name, Tagline, Email, Phone, Website, and Physical Address.
* **Visual Identity:**
    * **Logo Upload:** Support for custom brand logos.
    * **Color Palette:** Custom selection for Primary, Secondary, and Accent colors (HEX code support).
* **Live Preview:** A real-time sidebar component that visualizes how the brand looks as data is entered.

### 2. Asset Generation (Output)
A dashboard for creating specific marketing materials using the configured brand identity.
* **Asset Gallery:** A grid view of available document types.
* **Supported Formats:**
    * Business Cards (Styles: Modern, Classic, Minimal, Bold)
    * Invoices
    * Social Media Posts
    * Logo Mockups
    * Letterheads
    * Email Signatures
* **Customization:**
    * Real-time preview of assets with user data applied.
    * Style switching (e.g., swapping business card templates).
    * PNG Download functionality.

## Workflow
1.  **Define:** User inputs business details and visual preferences in the **Brand Setup** tab.
2.  **Select:** User navigates to the **Generate Assets** tab and chooses a category (e.g., Business Cards).
3.  **Refine:** User selects a specific design template (e.g., "Modern").
4.  **Export:** User downloads the final high-resolution asset.
